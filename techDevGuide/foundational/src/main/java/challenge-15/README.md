# Challenge-15 Edit Distance
## Description
Given two strings, write a function that determines the minimum edit distance between the two strings.  You can insert and modify characters.

### Example
>
```
String1 = "ABCD"
String2 = "ACBD"

Output 
2

Explaination
ABCD->ACCD->ACBD
```

>
```
String1 = "AC"
String2 = "ABCD"

Output
2

Explaination
AC->ABC->ABCD
```
